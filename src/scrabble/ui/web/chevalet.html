🎴 Mes lettres — cliquez une lettre pour la sélectionner, réarrangez librement

Cliquez une lettre pour la sélectionner, puis une case vide du plateau pour la poser. Pour réarranger vos lettres (réflexion, sans effet sur la partie) : cliquez une lettre puis un emplacement vide pour l'y déplacer, ou deux lettres pour les échanger. Clic droit sur une lettre : la renvoie vers le vide le plus proche de la fin. Les 2 emplacements vides servent à ce réarrangement. Une lettre déjà posée sur le plateau apparaît grisée et n'est plus sélectionnable tant qu'elle n'est pas retirée.
