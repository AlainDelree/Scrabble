    ✏️ Brouillon — réflexion libre, sans effet sur la partie

    Réarrangez librement vos 7 lettres sur les 9 emplacements pour trouver l'inspiration : cliquez une lettre puis un emplacement vide pour l'y déplacer, ou deux lettres pour les échanger. Clic droit sur une lettre : la renvoie vers le vide le plus proche de la fin. Les 2 emplacements vides (position libre) figurent une lettre déjà posée sur le plateau à prendre en compte ; aucune saisie possible.

    Vos lettres restent masquées tant que vous ne cliquez pas sur « Voir mes lettres ». Pensez à les cacher si un adversaire approche.
